# growi-plugin-attachment-viewer

GROWI の閲覧ページに添付されたファイルの一覧を表示・操作するプラグインです。

## 機能

- サイドバーのボタンから添付ファイル一覧モーダルを開く
- 添付ファイルをテーブル形式で表示
  - 画像プレビュー（サムネイル）
  - ファイル名・ファイル ID のコピー
  - ファイルサイズ・画像寸法の表示
  - Markdown リンク・URL のコピー
  - 個別ダウンロード
- 全ファイルを一括ダウンロード
- ダウンロード時のファイル名形式: `ファイル名-ファイルID.拡張子`

## インストール

GROWI の標準のプラグインインストール方法でインストールしてください。
登録後、ページを再読み込みすると有効になります。

## 使い方

ページを表示すると、サイドバーに **「添付ファイル」** ボタンが追加されます。
ボタンをクリックすると、そのページの添付ファイル一覧がモーダルで表示されます。

- **Esc キー** または画面外クリックでモーダルを閉じる
- ページ遷移時はモーダルが自動的に閉じる

## 動作要件

- GROWI v7.4.4 以降で動作確認済
- Navigation API をサポートするブラウザ（Chrome 102+ / Edge 102+ / Firefox 未対応）
